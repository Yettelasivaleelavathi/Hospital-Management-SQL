# 🏥 Hospital Management System - SQL

A Hospital Management System database project built using SQL. This project is designed to manage hospital operations such as patient records, doctor information, appointments, treatments, billing, and prescriptions through a relational database.

## 📌 Project Overview

The Hospital Management System helps in storing and managing hospital-related data efficiently. It demonstrates SQL concepts such as database creation, table relationships, joins, aggregate functions, subqueries, views, stored procedures, and triggers (if implemented). Hospital management databases commonly include entities such as patients, doctors, appointments, prescriptions, billing, and wards. :contentReference[oaicite:0]{index=0}
